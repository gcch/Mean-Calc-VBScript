# 平均計算スクリプト (VBScript)

## Description

中身が同形 (データ構成が同じ) である複数の CSV ファイルを、
各要素ごとに平均を取り、
1 つの CSV ファイルに書き出すだけの非常にシンプルなスクリプト。


## Requirements

VBScript で書かれているため、Windows 環境。
